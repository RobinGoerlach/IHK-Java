Circle class unit tests using jUnit4 


Installation jUnit 4
--------------------

* Download junit4 JAR file from https://junit.org/junit4/ (Project home page)
* Extract junit-4.13.2.jar from download and store it near the Project files
* In Eclipse right-Click on your project name then Click on Build Path and 
  select configure build path. Take  on libraries and click on the button 
  “Add External JARs”. In the upcoming window select the jar file from the 
  folder where you have saved your jar file. Click on Apply and Close. Done!
 
jUnit 4 usage 
-------------

Inside Eclipse right-click on JUnit Test Class file and select Run as JUnit Test

Create new unit test
--------------------

Inside Eclipse navigate to File/New/JUnit Test Case (Menu)
* Select New jUnit 4 Test
